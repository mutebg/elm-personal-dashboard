# Personal Dashboard

## Tech stack
Personal Dashboard written in Elm and NodeJS ( + firebase cloud functions )

## Functionality
- Instagram
  - last Instagram pictures

- Twitter
  - last tweets
  - last saved tweets

- Medium
  - last posts
  - last recommended

- Strava
  - last runs

- GitHub
  - last repositories

- RescueTime
  - activity for last days

- LastFM
  - recent tracks
  - weekly chart
  - weekly albums
  - weekly albums

- SetlistFM
  - last events
